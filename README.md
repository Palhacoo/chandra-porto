# Portofolio HTML/CSS (Minimal)

Ini adalah portofolio sederhana yang hanya memakai HTML dan CSS dengan beberapa animasi ringan.

Cara menjalankan

1. Buka folder proyek.
2. Buka `index.html` di browser (klik dua kali atau `Open with` -> browser).

Catatan

- Implementasi ini hanya demo tampilan & animasi; form tidak mengirim data.
- Jika ingin menambahkan gambar, letakkan file di folder `assets/` dan update path di `index.html`.
- File utama: `index.html`, `styles.css`.

Lisensi: bebas pakai untuk tujuan pembelajaran/portofolio.
